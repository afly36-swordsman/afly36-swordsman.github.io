随笔系列5：玩转计算机

Last updated on May 12, 2025 am

常用快捷键

  • win + E：打开文件资源管理器

  • win + I：打开设置

  • Ctrl + C/V/X/Z/S/A：复制/粘贴/剪切/撤销/保存/全选

  • win + Tab：切换/显示任务视图（配合鼠标点击）

  • Alt + Tab：自动切换窗口/标签

  • Ctrl + F：查找

  • win + L：锁定屏幕

  • win + D：回到桌面

  • F2 (Fn + F2)：重命名

  • F5 (Fn + F5)：刷新

  • F11 (Fn + F11)：全屏

  • Ctrl + B/U/I：加粗/下划线/斜体

  • Ctrl Shift N：新建文件夹

  • Ctrl + Home/End：光标快速移动到文件头/文件尾

  • Home/End：光标快速移动到这一行的开头/结尾

  • Shift + Home/End：当光标在尾部/头部时，选中一行内容

  • Tab / Shift Tab：右移/左移一个单元、一个缩进

快捷打开软件

创建快捷方式，放入一个文件夹内，将该文件夹路径加入环境变量，然后就可以通过 win + R 键入快捷方式名，打开一个APP。

win + R 键入：

  • control 控制面板

  • notepad 记事本

  • taskmg 任务管理器

  • calc 计算器

滑动关机

炫酷的滑动关机是系统自带的功能，可以通过：此电脑 -> C盘 -> Windows -> System32 文件夹里查找 SlideToShutDown.exe，然后创建桌面快捷方式，双击即可。

这里给出一个简单的方式：

  1. 新建快捷方式（桌面右键）

  2. 键入对象的位置（%windir\%System32\SlideToShutDown.exe）

  3. 改个喜欢的名字，换个喜欢的图标

  4. 双击即可滑动关机，也可固定到任务栏
